About CryptoLab

A modern cryptography learning platform

Purpose

CryptoLab is an educational web application designed to help students and enthusiasts learn about classical and modern cryptographic algorithms through hands-on experience with 15 different ciphers.

Tech Stack

This project demonstrates a complete CI/CD pipeline:

Python/Django GitHub Actions SonarCloud Docker Render

Available Ciphers

  • Substitution: Caesar, Multiplicative, Affine, Vigenere, Autokey
  • Transposition: Rail Fence, Columnar, Hill, Playfair
  • Stream: OTP, Vernam
  • Modern: Feistel, DES, AES, RSA
